Hey! Welcome to the Technology page!

This web application started as a group project with an effort to utilize technologies we learned in class along with some new technologies added in.

On the front end we implemented a new CSS framework called Materialize! It features clean and responsive designs. All the modals and buttons are key features of Materialize.

On the backend we made use of Socket.io a javascript library used to enable real time bi-directional communication between web clients and servers.
